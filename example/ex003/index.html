中国国界、省界及九段线

GMT示例 , 地学数据
刘珠妹
2016年10月22日

由于存在领土争议，GMT 自带的国界数据是无法直接用于发表的，因而用户需要自己准备国界数据。本文提供了网络上流传的几个国界相关数据，以及两个绘制中国地图的脚本。

数据下载： CN-border-La.dat CN-border-L1.dat CN-border-L2.dat nine-dash-line.dat

  1. 数据来源于网络，无法保证数据可通过审核；
  2. 欢迎用户共享更准确的国界数据；
  3. CN-border-La.dat 中包含最全的信息，包括国界、省界、九段线以及南海诸多小岛，但其中青海宁夏边界有误；
  4. CN-border-L1.dat 仅包含国界数据；
  5. CN-border-L2.dat 包含国界、省界以及南海诸多小岛，但南海诸岛信息不全；
  6. nine-dash-line.dat 仅包含九段线数据；

-JD 等距圆锥投影

#!/bin/bash
R=70/140/3/60
J=D105/35/36/42/7.5i
PS=CN-border-JD.ps

gmt set FONT_ANNOT_PRIMARY 9p
gmt set MAP_GRID_PEN_PRIMARY 0.25p,gray,2_2:1
gmt set FORMAT_GEO_MAP ddd:mm:ssF
gmt set MAP_FRAME_WIDTH 2p

gmt pscoast -J$J -R$R -G244/243/239 -S167/194/223 -B10f5g10 -Lg85/11+c11+w900k+f+u+l'Scale' \
    --FONT_ANNOT_PRIMARY=7p --FONT_LABEL=8p --MAP_LABEL_OFFSET=5p -K > $PS
gmt psxy CN-border-La.dat -J$J -R$R -W0.5p -O >> $PS
rm gmt.conf gmt.history

-JM 墨卡托投影

#!/bin/bash
R=70/138/13/56
J=M105/35/6.5i
PS=CN-border-JM.ps

gmt set FONT_ANNOT_PRIMARY 9p
gmt set MAP_GRID_PEN_PRIMARY 0.25p,gray,2_2:1
gmt set FORMAT_GEO_MAP ddd:mm:ssF
gmt set MAP_FRAME_WIDTH 3p

# 绘制中国地图
gmt pscoast -J$J -R$R -G244/243/239 -S167/194/223 -B10f5g10 -Lg85/17.5+c17.5+w800k+f+u+l'Scale' \
    --FONT_ANNOT_PRIMARY=7p --FONT_LABEL=8p --MAP_LABEL_OFFSET=5p -K > $PS
gmt psxy CN-border-La.dat -J$J -R$R -W0.5p -O -K >> $PS

# 绘制南海区域
R=105/123/3/24
J=M1.1i
gmt psbasemap -J$J -R$R -B0 -X5.4i --MAP_FRAME_TYPE=plain --MAP_FRAME_PEN=1p -K -O >> $PS
gmt pscoast -J$J -R$R -N1/0.1p -W1/0.25p -G244/243/239 -S167/194/223 -K -O >> $PS
gmt psxy CN-border-La.dat -J$J -R$R -W0.25p -O -K >> $PS
echo "SC Sea Islands" | gmt pstext -J$J -R$R -F+f10p+cBC -D0c/0.1c -N -Gwhite -O >> $PS

rm gmt.conf gmt.history
